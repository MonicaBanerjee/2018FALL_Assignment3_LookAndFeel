Website Design




Design – use of at least 3 specific design guidelines and ideas for web sites.

Consistency – the webpage design has been kept consistent across all tabs with respect to the layout and colors used.

Navigability – Having an intuitive navigation helps customers to go from point A to point B easily.

For that we have added various navigation buttons on the navigation bar to go view next and previous websites or go back to home. All the web pages are very well connected to each other.

Visual Hierarchy- It is like the simplicity design guideline but in this the web content is made in a way that the users are gravitated towards the content, for example “View Portfolio” is the first things the users would click on the index page before exploring anything else.

Simplicity- The simplicity is the most important guideline used in our webpage where not a single element is ambiguously designed and works intuitively.

Responsiveness



Used following tags for Responsive Web Design - max-width: 100% - margin: auto; - A viewport element gives the browser instructions on how to control the page's dimensions and scaling. Margin auto tag will set the relative size The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device).

Usability



ease of use and navigation of your interface end users We have created a basic and intuitive way of design for the end users which contains navigation menus and different buttons to redirect to the content where the user wants to. Have used hover functions and onclick function to display relevant information.

Content



Since we have made a portfolio website, the webpages display portfolios of the three group members extensively.
